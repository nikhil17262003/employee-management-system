# Employee Management System

A Full Stack Employee Management System built using:

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- HTML, CSS, JavaScript
- Postman

## Features
- Add employee
- View employee list
- Update employee details
- Delete employee
- REST API integration
- MySQL database

## API Endpoints
- GET /employees
- POST /employees
- PUT /employees/{id}
- DELETE /employees/{id}



## How to Run
1. Clone repository
2. Configure MySQL in `application.properties`
3. Run Spring Boot app
4. Open `http://localhost:8081`
